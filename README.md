# Agile Metrics and Flow

Practical delivery metrics and flow-based measurement frameworks
for enterprise Agile teams.

## Purpose
This repository focuses on metrics that improve delivery outcomes,
not vanity reporting.

## Coverage
- Flow metrics (throughput, cycle time, WIP)
- Predictability and delivery health
- Capacity-aware planning
- Common metric anti-patterns

## Intended Audience
- Agile Delivery Leaders
- Engineering Managers
- Program and Portfolio leaders
